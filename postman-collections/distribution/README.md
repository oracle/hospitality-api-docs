# OPERA Cloud Distribution Postman collections

Oracle Hospitality OPERA Cloud Distribution enables hotels to communicate room availability, rates, and inventory (ARI) directly to hotel chains, brands, independents, and representation companies.

The Postman collections use the [oracle-hospitality-distribution Postman Environment](oracle-hospitality-distribution.postman_environment.json).
